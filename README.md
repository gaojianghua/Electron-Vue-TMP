## 说明
Electron Vue TMP，是基于[Vue3](https://cn.vuejs.org/)框架与[Electron](https://www.electronjs.org/zh/)框架制作的桌面端应用模板。方便前端开发者快速搭建桌面端应用，节省开发周期，增加开发效率。
